How It Works

Forward Propagation: Data flows from input layer through hidden layer to output. Each connection has a weight that multiplies the signal.

Backpropagation: The error is calculated and propagated backward, adjusting weights to reduce the error.

Activation Function: We use the sigmoid function to introduce non-linearity, enabling the network to learn complex patterns.
